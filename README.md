# Spotify-Analysis
## Overview
This project explores Spotify user data to understand listening behavior, preferences, and usage patterns through data analysis and visualization.

## Features
Data cleaning and preparation
Exploratory analysis of user behavior
Visualization of trends and patterns
Key insights on preferences and usage

## Tech Stack
-Python 
-Pandas
-NumPy
-Matplotlib / Seaborn
-Jupyter Notebook

## Project Structure
Spotify-Analysis/
│── spotify_an.ipynb   
│── Spotify_data.xlsx            
│── README.md          

## What I Analyzed
Most users belong to 20-35 age group, indicating target audience concentration.
Certain genres like Melody and Classical dominate listening preferences, showing clear user taste patterns.
A significant portion of users use free plans, which could be an opportunity for premium conversion.
Recommendation ratings suggest average satisfaction levels.
Listening behavior peaks during night time slots, useful for targeted recommendations.

## How to Run

Clone the repository:

git clone ```https://github.com/seqyoi/Spotify-Analysis.git```

Navigate to the folder:

```cd Spotify-Analysis```

Install dependencies:

```pip install pandas numpy matplotlib seaborn notebook```

Run the notebook:

```jupyter notebook```
